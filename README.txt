The Senate Bus Problem

This project simulates the Senate Bus Problem using Java concurrency mechanisms.

Prerequisites
    - Java 11
    - Maven 3.x


How to use

    Build the project

        `mvn clean install`

    Run the project

        `java -jar target/the-senate-bus-problem-1.0-SNAPSHOT.jar <busArrivalMean> <riderArrivalMean>`

        Replace `<busArrivalMean>` and `<riderArrivalMean>` with appropriate values representing the mean arrival times in milliseconds.

        Example: `java -jar target/the-senate-bus-problem-1.0-SNAPSHOT.jar 1000 100`
